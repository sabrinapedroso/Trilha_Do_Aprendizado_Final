Exercícios JavaScript

Exercício 01

A partir de um núemro digitado pelo usuário determine se ele é par ou ímpar

Exercício 02

A partir de um núemro digitado pelo usuário determine se ele é divisível por 7 ou 3

Exercício 03

Faça um código que solicite ao usuário nome e idade. Apresente a idade "Bem vindo nome! Você tem xx anos.". Caso ele seja maior de 18 anos o programa deve solicitar o CPF e exibir o número do CPF. Caso contrário deve exibir uma mensagem dizendo que é menor de idade.
